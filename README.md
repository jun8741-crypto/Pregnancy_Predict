# Pregnancy Predict

## 실행

```bash
python final_submission_260424.py
```

`data/` 디렉터리에 `train.csv`, `test.csv`, `submission/sample_submission.csv`가 있어야 합니다.

- 주요 라이브러리: `lightgbm`, `catboost`, `scikit-learn`, `scipy`, `pandas`, `numpy`
- 선택(MODE='full'): `autogluon.tabular[all]`
